TTS_TYPE_IS¶

Required header: #include <tts/tests/types.hpp>

#define TTS_TYPE_IS( LHS, RHS )

Checks if LHS and RHS satisfy std::is_same_v<LHS,RHS> == true.

Parameters

  • LHS, RHS: Types to compare
