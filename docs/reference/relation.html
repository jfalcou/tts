Relational Tests¶

The following component provides macros to perform ordering test between values.

To use those macros, include the tts/tests/relation.hpp file.

Macros¶

Note

Ordering and equality comparisons are, by default, handled by the operator== or opetator< overloads. In the case one needs to perform a custom comparison, see the Custom Comparison section.

TTS_EQUAL(LHS, RHS)¶

Performs equality comparison between two values.

Required header: #include <tts/tests/relation.hpp>

Example:

#include <tts/tts.hpp>
#include <tts/tests/relation.hpp>

TTS_CASE( "Check correctness of equality tests" )
{
  TTS_EQUAL(45,45.f);

  std::vector<float> a{1,2,3}, b{1,2,3};
  TTS_EQUAL(a,b);
}
Parameters

  • LHS: Expression to compare

  • RHS: Expression to compare

Performs a less-or-equal-than comparison between two values.
