TTS_IEEE_EQUAL ---------------------------------------------------------------------------------------------------- **Synopsis:** **Required header:** `#include <tts/tests/precision.hpp>` Checks if two values are exactly within a 0 [ULP](rationale.html#precisiontesting) of each other. This also allow for infinites and NaNs to be compared equal if both values are the same infinites or are both NaN. This comparison is performed by using the proper `tts::ulpdist` overloads (see :ref:ulp). **Parameters:** : - `LHS`, `RHS`: Expressions to compare **Example:**
