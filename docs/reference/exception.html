Runtime Error Tests¶

The following component provides macros to perform basic tests on the throwing or absence of runtime exceptions. Note that those test macros requires the exception to be detected to inherits from std::exception.

To use those macros, include the tts/tests/exception.hpp file.

Macros¶

TTS_THROW(Expr, Exception)¶

Checks if a given expression throws a given exception type.

Required header: #include <tts/tests/exceptions.hpp>

Example:

void foo() { throw some_exception_type(); }

TTS_CASE( "Check that we can capture thrown exceptions" )
{
  TTS_THROW( foo(), some_exception_type );
}
Parameters

  • Expr: Expression to test

  • Exception: Exception type supposed to be thrown

TTS_NO_THROW(Expr)¶

Checks if a given expression throws no exception.

Required header: #include <tts/tests/exceptions.hpp>

Example:

void bar() noexcept { }

TTS_CASE( "Check that nothrow function are detected as such" )
{
  TTS_NO_THROW( bar() );
}
Parameters

  • Expr: Expression to test
