TTS_ALL_ULP_EQUAL ---------------------------------------------------------------------------------------------------- **Synopsis:** **Required header:** Checks if all values of two containers are within a given [ULP distance](rationale.html#precisiontesting) and if both arguments have the same size. One parameters can be set a scalar values, in which case the test will check the container parameter's size is 1. **Parameters:** : - `LHS`, `RHS`: Sequences to compare - `X`: Maximal ULPs acceptable for passing the test **Example:**
