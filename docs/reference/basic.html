Basic Tests¶

TTS_EXPECT¶

Required header: #include <tts/tests/basic.hpp>

#define TTS_EXPECT( Condition )

Checks if Condition evaluates to true.

Parameters

  • Condition: Expression to test

Example:

#include <tts/tts.hpp>
#include <tts/tests/basic.hpp>

TTS_CASE( "Check that expectation can be met" )
{
  int a = 42, b = 69;

  TTS_EXPECT(a != b);
  TTS_EXPECT(a <  b);
  TTS_EXPECT(a <= b);
  TTS_EXPECT(b >  a);
  TTS_EXPECT(b >= a);
}

TTS_EXPECT_NOT¶

Required header: #include <tts/tests/basic.hpp>

#define TTS_EXPECT_NOT( Condition )

Checks if Condition evaluates to false.

Parameters

  • Condition: Expression to test

Example:

#include <tts/tts.hpp>
#include <tts/tests/basic.hpp>

TTS_CASE( "Check that counter-expectation can be met" )
{
  int a = 42, b = 69;

  TTS_EXPECT_NOT(a == b);
  TTS_EXPECT_NOT(a >  b);
  TTS_EXPECT_NOT(a >= b);
  TTS_EXPECT_NOT(b <  a);
  TTS_EXPECT_NOT(b <= a);
}
