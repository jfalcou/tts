TTS_ALL_IEEE_EQUAL ---------------------------------------------------------------------------------------------------- **Synopsis:** **Required header:** Checks if all values of two containers are exactly within a 0 [ULP](rationale.html#precisiontesting) of each other and if both arguments have the same size. This also allow for infinites and NaNs to be compared equal if both values are the same infinites or are both NaN. One parameters can be set a scalar values, in which case the test will check the container parameter's size is 1. **Parameters:** : - `LHS`, `RHS`: Sequences to compare - `...`: Optional tag.
