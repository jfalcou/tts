Status tests¶

The following component provides macros to display informations about tests status within scenarios.

To use those macros, include the tts/tests/infos.hpp file.

Macros¶

TTS_PASS(Message)¶

Register a test which always passes and streams a custom Message string.

Required header: #include <tts/tests/infos.hpp>

Example:

#include <tts/tts.hpp>
#include <tts/tests/infos.hpp>

TTS_CASE( "Check that forced pass passes" )
{
  TTS_PASS("Forced pass");
}
Parameters

  • Message: A literal string to display as additional informations

TTS_FAIL(Message)¶

Register a test which always fails and streams a custom message string.

Required header: #include <tts/tests/infos.hpp>

Example:

#include <tts/tts.hpp>
#include <tts/tests/infos.hpp>

TTS_CASE( "Check that forced failure fails" )
{
  TTS_FAIL("Forced fail");
}
Parameters

  • Message: A literal string to display as additional informations

TTS_INVALID(Message)¶

Register a test as invalid and streams a custom message string.

Required header: #include <tts/tests/infos.hpp>

Example:

#include <tts/tts.hpp>
#include <tts/tests/infos.hpp>

TTS_CASE( "Check forcing a test to be invalid" )
{
  TTS_INVALID("Forced invalid");
}
Parameters

  • Message: A literal string to display as additional informations
