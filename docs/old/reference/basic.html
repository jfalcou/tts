Basic Tests¶

The following component provides macros to perform basic tests over expression that simply evaluates as a boolean value.

To use those macros, include the tts/tests/basic.hpp file.

Macros¶

TTS_EXPECT(Expr)¶

Checks if a given expression evaluates to true.

Required header: #include <tts/tests/basic.hpp>

Example:

#include <tts/tts.hpp>
#include <tts/tests/basic.hpp>

TTS_CASE( "Check that expectation can be met" )
{
  int a = 42, b = 69;

  TTS_EXPECT(a != b);
  TTS_EXPECT(a <  b);
  TTS_EXPECT(a <= b);
  TTS_EXPECT(b >  a);
  TTS_EXPECT(b >= a);
}
Parameters

  • Expr: Expression to test

TTS_EXPECT_NOT(Expr)¶

Checks if a given expression evaluates to false.

Required header: #include <tts/tests/basic.hpp>

Example:

#include <tts/tts.hpp>
#include <tts/tests/basic.hpp>

TTS_CASE( "Check that counter-expectation can be met" )
{
  int a = 42, b = 69;

  TTS_EXPECT_NOT(a == b);
  TTS_EXPECT_NOT(a >  b);
  TTS_EXPECT_NOT(a >= b);
  TTS_EXPECT_NOT(b <  a);
  TTS_EXPECT_NOT(b <= a);
}
Parameters

  • Expr: Expression to test
