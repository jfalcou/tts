Scenario definition ==================================================================================================== The following component provides macros to define test cases as a simple function or as a template function. Those are the basic blocks of writing tests with TTS.
To use those macros, include the `tts/case.hpp` file. Status Tests ==================================================================================================== The following component provides macros to display informations about tests status within scenarios.
To use those macros, include the `tts/tests/infos.hpp` file. To use those macros, include the `tts/tests/basic.hpp` file. Relational Tests ==================================================================================================== The following component provides macros to perform ordering test between values.
To use those macros, include the `tts/tests/relation.hpp` file. !!! Attention Note Ordering and equality comparisons are, by default, handled by the `operator==` or `opetator<` overloads. In the case one needs to perform a custom comparison, see the :ref:`custom-comparison` section. Precision Tests ==================================================================================================== The following component provides macros to perform precision tests between values.
To use those macros, include the `tts/tests/precision.hpp` file. !!! Attention Note Precision comparisons are, by default, handled by the `tts::ulpdist`, `tts::reldist` or `tts::absdist` overloads. In the case one needs to perform a custom precision comparison, see the :ref:`custom-precision` section. Sequence Tests ==================================================================================================== The following component provides macros to perform tests over sequence of values. All those macros are based on the fact that the check to perform must pass for every elements of the sequence.
For convenience, one can substitute one of two of the input sequences by a scalar that will be interpreted as a sequence of one element.
To use those macros, include the `tts/tests/sequence.hpp` file. !!! Attention Note Precision comparisons are, by default, handled by the `tts::ulpdist`, `tts::reldist` or `tts::absdist` overloads. In the case one needs to perform a custom precision comparison, see the :ref:`custom-precision` section. Runtime Error Tests ==================================================================================================== The following component provides macros to perform basic tests on the throwing or absence of runtime exceptions.
Note that those test macros requires the exception to be detected to inherits from `std::exception`.
To use those macros, include the `tts/tests/exception.hpp` file. Type Tests ==================================================================================================== The following component provides macros to perform tests over expression type equivalence or properties.
To use those macros, include the `tts/tests/types.hpp` file.
